--Jetons fous par Samy Bahi et Théophile Billod--

Jeton.java => contient l'objet jeton, définit par une position et une direction (et une booléen d'existence)
Damier.java => contient l'objet damier, consititué d'une hauteur, d'une largeur et d'un tableau de jetons, ainsi que de quelques autres attributs utiles pour le fonctionnement du programme
Les constantes utilisées pour le damier et l'affichage graphique sont stockées dans la classe damier.
JetonsFous.java => Le main, contient l'interface utilisateur.

Compilez Jeton.java, puis Damier.java, et enfin JetonsFous.java.

Exécutez ensuite JetonsFous.java (le main), et entrez les paramètres demandés dans la console (partie prédéfinie, dimensions du damier, nombre de jetons).
(une saisie invalide provoquera un refus du paramètre et une nouvelle demande de saisie)

Le programme s'exécute automatiquement une fois les paramètres entrés, avec un affichage console et un affichage graphique dans une nouvelle fenêtre.

Le travail a été réalisé en collaboration directe avec Bahi Samy, nous travaillions et réfléchissions ensemble sur le même ordinateur.
